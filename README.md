# OrderManagementSystem

This is a deliberately flawed .NET 8 Web API project for Copilot review and improvement suggestions.

## Features
- Order, Customer, Product management
- Manual DTO mapping
- Repository, Service, and Data layers
- Architectural and code quality issues for Copilot to identify

## Known Issues
- Repository pattern violations
- SOLID principle violations
- Naming convention problems
- Manual DTO mapping
- Missing validation
- Poor documentation
- Performance anti-patterns
- Magic numbers
- Missing dependency injection configuration

## How to Run
1. Install .NET 8 SDK
2. Run `dotnet run` in the project directory

## Improvement Opportunities
- Implement proper repository and service abstractions
- Use AutoMapper and FluentValidation
- Refactor for SOLID principles
- Add documentation and optimize code
